Color Libraries

Color libraries are small CSS files containing commonly used color values like safety standards, highway colors, fashion colors, army camouflage colors, and more.

Highway Yellow

Color libraries are very easy to use. Just add a library link to your web page:

<link rel="stylesheet" href="http://www.w3schools.com/lib/w3-colors-highway.css">

And a class name to the HTML element you want to color:

<div class="w3-highway-yellow">Highway Yellow
